## My Gallery web app using MEAN stack
#### MongoDb, Express.js, Angular, Nodejs
#### Other techniques: Cloudinary, TypeScript
#### Deploy with Heroku

#### Quick start 
`npm i -g`

`npm i -g @angular/cli`

`cd angular-src`

`npm i -g`
#### For server testing localhost:5000
`npm run server`
#### For whole application testing
`npm run server`

`cd angular-src`

`ng serve --open`
